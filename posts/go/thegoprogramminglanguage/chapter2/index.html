名称

  • 在Go中使用驼峰式来命名名称（大驼峰或小驼峰都可以），名称区分大小写，并且名称的开头应该是一个字母或者下划线，后面可以是任意的字符、数字、下划线。
  • 实体 第一个字母大写表示对包外是可见的和可访问的。

声明

在Go中只有四个主要的声明：变量（var），常量（const），类型（type），函数（func）。

Go文件格式

  1. 一个Go文件应该有一个package在文件开头表明文件属于那个包。
  2. 接着是一个import声明，表示导入的包。
  3. 接下来是包的级别的类型，变量，常量，函数的声明。
package main

import "fmt"

//(包级别的声明) 不仅只对这个文件可见，而是对同一个包中的所有源文件都可以见
const bollingF = 212.0

func main(){

	var f = bollingF
	var c = (f - 32) * 5 / 9
	fmt.Printf("boiling point = %g or %g°C\n", f, c)
}

变量

var name type = expression

我们可以将type或者expression省略一个，如果省略type，则可以从expression中推断出，如果省略expression，则Go会对他自己进行初始化，数字为0，布尔类型为false，字符串为“”，接口和其他的引用类型为nil。

var a, b, c int
var x, y, z = 1, 2, 3

短变量声明

我们要知道短变量是用来声明和初始化局部变量。

name := expression

如果短变量声明的左边变量在之前被定义，那么他的行为就跟复制一样，但是每一个短变量声明中都必须声明一个新变量

var a int
a, b := 1, 2

var x, y int
x, y := 3,4 // 错误，不存在一个新变量

指针

在C语言中我们就学过指针，指针是一种特殊的变量，他用来存储一个变量的地址，变量的地址可以被设置为许多指针变量的地址。

var a *int  // 定义一个指针变量
x := 1
a = &x // 或组一个变量的地址
println(*a) // 获取指针指向的值
*a = 2 //  更改指针指向的值
println(*a)

指针的零值是nil，只有当指针变量指向同一个变量和同时等于nil时才相等。

new 函数

new函数用来创建一个变量，new(T)初始化一个T类型的零值，返回地址（*T）。每次调用new都是不同的地址，可以理解使用new就是在内存中分配一块内存，这个内存的地址当然是不同的。

p := new(int)
fmt.Println(*p)
*p = 2;
fmt.Println(*p)

变量生命周期

变量的生命周期是指比变量在程序执行过程中存在的时间段，毫无疑问，包级别的变量在整个程序的运行周期内都是有效的，而局部变量有一个动态生命周期，直到他变得不可访问时，他的生命周期结束，内存被回收。局部变量在函数内分配内存在堆上而不是在栈上，Go称为逃逸。

var gobal *int

func f() {
	x := 1
	gobal = &x // x进行逃逸
}

变量存储的位置有栈或者堆上，但是这个并不是变量所决定，而应该有变量的作用域来决定。

赋值

赋值就不用说了，毕竟不是第一次学编程，但是Go支持多重赋值。

var y, z int
y, z = 1, 2 // 多重赋值
fmt.Println(y, z)
y, z = z, y // 多重复制可以实现数据的交换
fmt.Println(y, z)

类型声明

我们可以为一个Go的底层类型定义一个新的类型名称，他通常是一个包级别的声明。

type name underlying-type

// 新类型
type Celsius float64 // 可导出，首字母大写
type Fahrenheit float64

func CToF(c Celsius) Fahrenheit { return Fahrenheit(c*9/5 + 32) }
func FToC(f Fahrenheit) Celsius { return Celsius((f - 32) * 5 / 9) }

每一个被定义的类型T都会一个T(x)方法将x类型转成T类型，如这里的Celsius(x)方法（这里的x是float64）。

var c1 Celsius = 646
var f1 Fahrenheit = 131
fmt.Println(c1, f1)
// c1, f1 = f1, c1 // 报错，类型不一样
c1, f1 = FToC(f1), CToF(c1)
fmt.Println(c1, f1)

类型别名

type intalias = int

类型别名是给类型取一个别称，但是他跟原来的底层类型都是一样的用法，可以相互替换，并不是一种新的类型。[更多][https://polarisxu.studygolang.com/posts/go/action/type-and-alias/]

type inialias = float64
func main(){
	var a1 float64 = 23.9
	var a2 inialias = 32.0
	fmt.Println(a1, a2)
	a1, a2 = a2, a1
	fmt.Println(a1, a2)
}

包

这个没看懂 Go中将在工作区中的一个文件夹叫做包，一个包中保证若干个.go文件。每一文件的开头使用package来声明一个包的名称，package声明前面是文档注释，通常使用一句话对包进行总结。

包的导入

在Go中包通过一个叫做导入路径的唯一字符串来标识，导入路径是一个目录，目录中有许多的Go源文件，而包名就是导入路径的最后一段。

包的初始化

包的初始化从包级别的变量开始，按照声明的顺序初始化，而且在编译器编译的时候，会对所有的文件进行一个排序，同时go会保证最需要的会被最先初始化，最后初始化main函数。同时有一些初始化的变量很难有简单的初始化表达式来初始化，我们可以使用init函数来进行初始化。

package main

import (
	"fmt"
)

var a = b + c 
const b = 1
var c = 2

var array [10]int

func init(){
	for i := range array{
		array[i] = 3
	}
}

func main(){
	fmt.Println(a, b, c)

	for i := range array{
		fmt.Println(array[i])
	}
}
