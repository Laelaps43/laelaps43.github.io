前言

复合数据类型由基本的数据类型已各种方式组合而成。 四种复合数据类型：数组、slice、map、结构体。 数组中的元素具有同一种类型，而结构体中没有这种要求，slice、map是动态数据结构。

数组

数组是拥有固定长度的有系统数据类型的元素序列，他跟我们其他语言的数字是一样的。

package main

import "fmt"

func main() {
	var a [3]int // 创建一个长度为三的数组，默认填充零值
	a1 := [3]int{}
	a2 := [3]int{1, 2, 3} // 创建一个为三的数组，给定值
	a3 := [3]int{1, 2}	// 为指定的为值为零值
	a4 := [...]int{1, 2, 3, 4} // 数组的长度由初始化元素的个数决定
	a5 := [...]int{50: -5, 99: 5} // 数组的长度为一百，下标50为-5,下标99为5
	fmt.Println(a[1])
	fmt.Println(a1[0])
	fmt.Println(a2[0])
	fmt.Println(a3[2])
	fmt.Println(a4[2])
	fmt.Println(len(a4))
	fmt.Println(a5[50], a5[99])	
	fmt.Println(len(a5))
	fmt.Printf("%t\n", a == a1) // 如果数字的类型和数组的基本数据类型可比较，则可比较
	fmt.Printf("%t\n", a2 != a3)
}

Slice

Slice是一个有相同类型的可变长度的序列，Slice之间不能比较；slice唯一比较的就是跟nil比较，但是不能根据slice是否为nil来判断slice是否为空，因为有可能长度为空，但是拥有底层数组。

属性

  1. 指针：指向数组中可以被slice访问的第一个元素
  2. 长度：slice元素个数，使用len函数返回
  3. 容量：slice中可容纳元素的大小，通常是slice的起始元素到数组的最后一个元素之间的个数，使用cap函数返回

创建

// 底层数组是自己创建的
a1 := [...]int{1, 2, 3, 4}
p1 := &a1
s1 := a1[:2] // 等于a1[0:2]
s2 := p1[1:] // 等于a1[1:len(a1)]
fmt.Printf("%q\n", s1) 
fmt.Printf("%q\n", s2)

var s3 []int // 声明一个slice，len(s) == 0 s == nil

// 使用make创建
s4 := make([]int, 2, 4) // 长度为2，容量为4
s5 := make([]float64, 2) // 长度=容量=2
fmt.Printf("长度为%d，容量为%d。\n", len(s4), cap(s4))
fmt.Printf("长度为%d，容量为%d。\n", len(s5), cap(s5))

添加元素

使用append添加元素，但是需要使用原来的slice去接受返回来的返回值，一是因为我们添加一个参数有可能会导致底层数组的大小调整，所有对底层数组的长度或者容量，再者将slice指向不同的底层数组都需要更新slice，二是因为更新了一个slice指针，长度和容量有需要更改，因为他们是需要显示赋值的。

s3 := []int{1, 2, 3, 4}
fmt.Printf("长度为%d，容量为%d。\n", len(s3), cap(s3))
s3 = append(s3, 5)
fmt.Printf("长度为%d，容量为%d。\n", len(s3), cap(s3))
s3 = append(s3, 6, 7)
fmt.Printf("长度为%d，容量为%d。\n", len(s3), cap(s3))
s3 = append(s3, s3...) // 追加s3中的所有参数，因为函数可以接受一个可变长参数列表
fmt.Printf("长度为%d，容量为%d。\n", len(s3), cap(s3))

stack := []int{} // 模拟一个栈
stack = append(stack, 1) // push
stack = append(stack, 2) // push
stack = append(stack, 3) // push
stack = stack[:len(stack) -1] // pop
top := stack[len(stack) - 1]
fmt.Println(top)

Map

Map是一个种拥有键值对的无序集合，键是唯一的，可以根据键来获取、更新和移除对应的键值对。 对于Map的键类型和值类型不一定相同，但是键之间必须是可以通过==来比较的类型，但是map类型本身是不可比较的。

// 使用make创建一个map
person1 := make(map[string]int)
person1["Ron"] = 12 // 往map中添加一对键值
person1["Lae"] = 22
// 使用字面量创建一个带初始化的map
person2 := map[string]int{
	"Ron": 12,
	"Lae": 22,
}
// 根据键删除对应的键值对
delete(person2, "Lae")
// 创建空的map
person3 := map[string]int{}
person3["Ron"] = 12
fmt.Println(person3["Lae"]) // 不存在的键值对返回值的零值

我们可以使用for配合range来遍历所有的键和对应的值；但是迭代的顺序是随机的，如果我们需要对顺序来遍历，我们需要对键进行排序，然后在从map中获取值。

person2 := map[string]int{
	"Ron": 12,
	"Lae": 22,
}
for k, v := range person2{
	fmt.Printf("%s\t%d\n", k, v)
}
keys := make([]string, 0, len(person2))
for k := range person2 {
	keys = append(keys, k)
}
sort.Strings(keys)
for _,k := range keys {
	fmt.Printf("%s \t %d\n", k, person2[k])
}

在使用map时必须初始化，通过下标（尽管这个下标不存在）访问时总是有值，他会返回值类型的控制，那么怎么判断一个一个元素是否存在或者这个元素的值恰好是0呢！使用下标访问时会返回两个变量，第二个值用来报告这个值是否存在。

person2 := map[string]int{
	"Ron": 12,
	"Lae": 22,
}
if v1, ok := person2["pss"]; !ok{
	fmt.Println("pss not exist")
}else{
	fmt.Println(v1)
}

Go没有人提供集合，但是我们可以使用map来实现这一功能。

package main

import (
	"bufio"
	"fmt"
	"os"
)

//!+
func main() {
	seen := make(map[string]bool) // a set of strings
	input := bufio.NewScanner(os.Stdin)
	for input.Scan() {
		line := input.Text()
		if !seen[line] {
			seen[line] = true
			fmt.Println(line)
		}
	}

	if err := input.Err(); err != nil {
		fmt.Fprintf(os.Stderr, "dedup: %v\n", err)
		os.Exit(1)
	}
}

结构体

结构体是一种聚合数据类型，他将零个或者多个任意的类型的变量组合在一起形成。结构体中的每一个变量都叫作结构体的成员。

type Employee struct{
	ID int
	Name, Address string
	DoB time.Time
	Positon string
}

结构体中的成员变量的顺序很重要，如果不是系统的顺序就不是系统的类型；成员变量名称首字母大写表示变量是可导出的，当然你也可以不导出。 结构体中不能包含有结构体本身的变量，但是可以包含结构体本身的指针变量，这个也好理解，指针是一个一种类型，只是表示了属于那一种类型的指针。 结构体的零值有成员变量的零值所确定，结构体也可以没有任何的成员变量，叫作为空结构体struct{}。 结构体使用点来访问成员

var employee Employee
employee.ID = 1
// 当然我们可以获取结构体中变量的指针
name := &employee.Name
*name = "Ron"
// 结构体指针
emp := &employee
emp.Address = "abc"
// 等于
(*emp).Address = "abc"

字面量

我们可以通过设置结构体的字面量来设置成员变量的值；而且我们大部分都是使用结构体的指针来传递给函数或者从函数中返回。

  • 安装结构体中的顺序来为每个成员变量设置一个值，但是不常用。 
    type Student struct{
	ID int
	Name string
}
student := Student{1, "Ron"}
// student := &Student{1, "Ron"}
fmt.Println(student)
  • 通过指定部分或者全部成员变量的名称和值来初始化变量，没有指定的为变量类型的默认零值 
    student1 := Student{ID: 1, Name: "Ron"}
// student1 := &Student{ID: 1, Name: "Ron"}
fmt.Println(student1)

注意：

package p
t ype T struct { a, .b int } // a 和b 都是不可导出的

package q
import "p"
var _ = p.T{a : 1, b: 2} ／／ 编译错误， 无法引用a 、b
var _ = p.T{ l, 2} ／／ 编译错误，无法引用a、b

结构体比较

如果结构体中的成员变量是可比较的，则结构体是可比较的，还可以作为结构体的键类型。

结构体嵌套&匿名成员

在结构体中可以将另外一个结构体嵌套到结构体中，但是访问被嵌套的结构体需要多层访问；而Go中可以定义不带名称的结构体成员（匿名成员），只指定结构体的类型，我们可以直接的访问，而不用多次嵌套的访问。

type Person struct{
	Name string
	age int
}

type Student struct{
	ID int
	Person Person
}

var stu Student
stu.ID = 1
stu.Person.age = 18
stu.Person.Name = "Ron"
fmt.Printf("%#v\n", stu)

// 匿名嵌套
type Student1 struct{
	ID int
	Person
}

var stu1 Student1
stu1.ID = 2
stu1.age = 20
stu1.Name = "Lae"
fmt.Printf("%#v\n", stu1)

使用匿名成员的结构体字面量定义

stu2 := Student{1, Person{"Ron", 1}}
fmt.Printf("%#v\n", stu2)

stu3 := Student1{1, Person{"Lae", 20}}
fmt.Printf("%#v\n", stu3)

结构体标签

结构体在定义的时候处理字段和类型之外，还可以定义一个可选的tag标签。

struct结构体在成员变量后面使用[“”]或者[``]定义一些描述性的信息。

Tag在结构体字段后的格式：

`key1:"value1" key2:"value"`
  • key为非空字符串，字符串不能包含控制字符、空格、引号、冒号
  • value为双引号标记的字符串
  • Tag可以由多个键值对组成，键值对之间使用空格分隔
  • 如果当一个value中有多个值，有的使用逗号风格，不同的库中的实现是不一样的
type User struct { 
	Id int `json:"id" gorm:"column:id;primaryKey"` 
	Name string `json:"name"` 
	Password string `json:"password,omitempty"` 
}
  • “-”：表示不需要解析这个字段
  • “omitempty”：当字段为空(默认值)时，不要解析这个字段；比如是false、0、nil或者长度为0的array、map、slice、string等。
  • FieldName：当解析json、xml、ini等的时候用这个名字。
