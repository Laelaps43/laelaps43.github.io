类型分类

Go语言的数据类型有四类：基础类型（basic type）、聚合类型（aggregate type）、引用类型（reference type）、接口类型（interface type）。

整数

在Go中有有符合整数和无符合整数两种。 有符合整数：int8（8位）、 int16 （16位）、int32（32位）、 int64（64位） 无符合整数：uint8（8位）、 uint16 （16位）、uint32（32位）、 uint64（64位） 此外还有int和uint两种类型，他们在不同的平台有不同的大小，他等于平台上运算效率最高的值。 rune类型是int32类型的同义词，他用来代表这个值是一个Unicode码点，bete类型是uint8类型的同义词，他表示这个值是一个原始数据。 uintptr是一种无符号整数类型，他的大小并不明确，但是他可以存放一个完整的指针。 关于其他对整数的操作跟其他语言差不多

浮点数

在Go中的浮点数值只有float32和float64。

复数

在Go中有两种复数类complex64和complex128，他们分配由float32和float64构成。在Go中有complex函数根据实部和虚部来创建一个复数，使用real和imag函数来分别提取实部和虚部。 复数可以使用== 和!=来判断是否相等。

c1 := complex(1, 2)
c2 := complex(2, 1)
fmt.Println(c1)
fmt.Println(real(c2))
fmt.Println(imag(c2))
fmt.Println(c1 == c2)

布尔值

跟其他编程语言差不多

常量

常量是一种在编译阶段就可以计算出值的表达式；常量的声明定义了具体的值，他在运行阶段不可被更改。

const (
	pi = 3.1415946
	e = 2.78
)
// 或者指定类型
const (
	b float64 = 2.3333
	c int = 2
	d bool = true
)

如果我们声明一组常量时，只写第一项，那么其他项会服用前一项。

const (
	x = 1
	y // 1
	z = 2
	r // 2
)

iota

我们可以使用常量生成器iota生成一些列相关的值，他从0开始，每次加一。

const (
	a = iota
	b
	c
	d
)

const (
	a1 = 1 << iota
	b1
	c1
	d1
)

const (
	a2 = 1 + (2 * iota)
	b2
	c2
	d2
)

无类型常量

没有显示或者隐式声明其类型的常量,在一些特定的场景下编译器会自动赋予合适的类型，他们有着更高的精度。无类型常量之间可以进行计算；并且如果将常量转换成另外一种类型，则要求目标类型能够容纳原值。 常量可以是有类型的，或者无类型的。字面值常量， true , false , iota 以及一些仅包含无类型的恒定操作数的常量表达式是无类型的。

字符串

字符串是不可变的字节序列，意味着多个字符串可以使用同一段内存，复制字符串开销及其低廉。子串和字符串使用的是同一个数据。 在Go中字符串被解读为UTF-8的Unicode码点，而使用len函数返回的是字节数，而不是通常编程语言里返回的字符的数目。 子串生成的操作为使用s[i:j]来生产一个子串，包含i但是不包含j。

字符串字面量

字符串的值可以直接使用带有双引号的字节序列，可以是我们可读的字符等，也可以是Unicode码点或者是十六进制、八进制。 我们可以使用`…`来表示原生的字符串字面量；在他当中的转义部起作用，并且可以换行。

j := `abc
		\ 123 \n 
		abc`
fmt.Println(j) // 原封不动的打印出来

其他类型转换字符串

L3 := string(0x9EC4)

strings包

func HasPrefix(s, prefix string) bool

HasPrefix用来判断给定的字符串中是否以指定的前缀开头

s := "Hello, 世界"
prefix := "He"
fmt.Printf("%t\n", strings.HasPrefix(s, prefix))

func HasSuffix(s, suffix string) bool

用来判断是否以指定的字符串结尾

s := "Hello, 世界"
suffix := "lae"
fmt.Printf("%t\n", strings.HasSuffix(s, suffix))

func Contains(s, substr string) bool

判断是否包含指定的字符串

s := "Hello, 世界"
contain := "lo"
fmt.Printf("%t\n", strings.Contains(s,contain))

func LastIndex(s, substr string) int

返回最后出现的子串位置，如果不存在返回-1。

s := "go go goto"
fmt.Println(strings.LastIndex(s, "go"))
fmt.Println(strings.LastIndex(s, "og"))
fmt.Println(strings.Index(s, "go"))

func Index(s, substr string) int

返回子串第一次出现的位置，不存在返回-1。

func Count(s, substr string) int

计算字符串中子串出现的个数，如果substr是空的，返回 1 + sub中码点的数量

fmt.Println(strings.Count("cheese", "e"))
fmt.Println(strings.Count("黄小黄", ""))

func Fields(s string) []string)

将s中被空白符隔开的字符分隔出来，返回一个slice，如果字符串是空的，返回一个空的slice。

fmt.Printf("Fields are: %q\n", strings.Fields("  foo bar  baz   "))
fmt.Printf("Fields are: %q\n", strings.Fields(""))
fmt.Printf("Fields are: %q\n", strings.Fields(" "))
fmt.Printf("Fields are: %q\n", strings.Fields("foo"))

utf8包

RuneCountInString

用来返回字符串中有多少个rune

s := "Hello, 世界"
fmt.Println(utf8.RuneCountInString(s))
// 等于
count := 0
for i:= 0; i < len(s);{
	r, size := utf8.DecodeRuneInString(s[i:])
	fmt.Printf("%d\t%c\n", i, r)
	count += 1
	i += size
}
fmt.Println(count)
// 或者等于
count1 := 0
for i, r := range(s){
	fmt.Printf("%d\t%c\n", i, r)
	count1 += 1	
}
fmt.Println(count1)
// 再或者
count2 := 0
for range(s){
	count2 += 1	
}
fmt.Println(count2)

DecodeRuneInString

返回给定字符串的第一个rune，返回文本字符本身以及rune占用的字节数。

strconv包

func Itoa(i int) string

Itoa（integer to ASCII）将一个数值转换成字符串类型。 或者采用fmt.Sprintf

in := 123
fmt.Println(strconv.Itoa(in))
fmt.Println(fmt.Sprintf("%d", in))

func FormatInt(i int64, base int) string

将一个整数转化成给定进制的字符串，2<=base<=36。

v := int64(123)

s10 := strconv.FormatInt(v, 10)
fmt.Printf("%T, %v\n", s10, s10)
	
s16 := strconv.FormatInt(v, 16)
fmt.Printf("%T, %v\n", s16, s16)

func FormatUint(i uint64, base int) string

跟FormatInt作用一样，只是转化的是无符号整数。

func Atoi(s string) (int, error)

将一个字符串转换成一个整数。

v := "10"
if s, err := strconv.Atoi(v); err == nil {
	fmt.Printf("%T, %v", s, s)
}

func ParseInt(s string, base int, bitSize int) (i int64, err error)

将一个字符串安装指定的进制和指定的大小转换成一个整数，字符串在前面可以带有正负号，或者带有”0b"表示二进制，“0”或者“0o"表示八进制，”0x“表示十六进制，否则为十进值，如果给定的base为0，默认为他的前缀表示。
