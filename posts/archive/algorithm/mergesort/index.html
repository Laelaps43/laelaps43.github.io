归并排序是一次采用分而治之算法解决排序的问题，他是一种高效的，可以产生稳定的排序方法。

主要思想

归并排序是一中分而治之的算法，他将一个大问题不断分解成一序列小问题，在所有的小问题中寻求最优解。 在排序的过程中，我们将一个大的数组，分解成两个较小的子数组，而在将这两个子数组在次进行递归划分，知道子数组长度为1（此时我们认为在他以及是有序的），然后对子数组进行排序。 排序示意图：

性能分析

算法的时间复杂度在最坏，最好，平均复杂度都为O(nlogn)，而空间复杂度为O(n)（可以使用空间复杂度为O(1)）。

算法实现

归并

我们要将两个已排序的子数组进行归并，其实就是将两个子数组在次合并成一个大的已经排序好的数组。这里我们需要一个辅助数组来存储将要归并的数组，通常这个辅助数组的大小通常为要排序数组的大小。我们将要排序的子数组全部拷贝到辅助数组，然后再将排序好的数据复制到要排序的数组。

 /**
     * 排序归并数组
     * 归并 a[lo .. mid] 和 a[mid + 1 .. hi]
     */
    private static void merge(Integer[] a, int lo, int mid, int hi){
        int i = lo, j = mid + 1;

        // 复制数组到辅助数组，因为源数组会被排序结果覆盖
        if (hi + 1 - lo >= 0) System.arraycopy(a, lo, aux, lo, hi + 1 - lo);

        for (int k = lo; k <= hi; k++){ // 需要比较k次
            // 左半边用尽，将右半边剩余的数据依次放入
            if ( i > mid)               a[k] = aux[j++];
            // 右半边用尽，将左半边剩余的数据依次放入
            else if ( j > hi)           a[k] = aux[i++];
            // 左半边首字符小于右半边字符，将左半边字符放入
            else if ( aux[i] < aux[j])  a[k] = aux[i++];
            // 左半边首字符大于右半边字符，将右半边字符放入
            else                        a[k] = aux[j++];
        }
    }

自顶向下（递归）

他将数组不断的分解，直到分解到大小为1。然后对左半部分进行排序，然后在对右半部分进行排序，最后将左右两部分合并在一起，不断的递归后完成对数组的排序。 

public static void sortByRec(Integer[] a){
        aux = new Integer[a.length];
        sortByRec(a, 0, a.length -1);
    }

    private static void  sortByRec(Integer[] a, int lo, int hi){
        // 长度为1
       if (hi <= lo)  return;
       // 左半部分终点
       int mid = lo + (hi - lo)/2;
       // 对左半部分进行递归
        sortByRec(a, lo, mid);
        // 对右半部分进行递归
        sortByRec(a, mid + 1, hi);
        // 然后对左右半部分排序好的子数组进行归并
        merge(a, lo, mid, hi);
    }

自低向上（迭代）

在自低向上的算法中，我们首先在数组中取长度为1的子数组，而在下一轮的子数组大小因为为前一个子数组的两倍（2 * subArray），然后在对两个子数组进行排序。值得注意的是在每一次的改变子数组的大小时，都需要在全部数组中对两个长度为subArray的子数组进行一次归并。

    public static void sortByIteration(Integer[] a){
        int length = a.length;
        aux = new Integer[length];
        for (int size = 1; size < length; size *= 2){  // 子数组大小
            for (int lo = 0; lo < length - size; lo += (size * 2)){ // 子数字的索引
                // 因为最后一个数组并不可能会会是 size大小，他有可能 lo + (size * 2) >  length
                merge(a, lo, lo + size - 1, Math.min(lo + (size * 2) - 1, length - 1));
            }
        }
    }
