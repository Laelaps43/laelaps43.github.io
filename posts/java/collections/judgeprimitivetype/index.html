我们需要判断当前数据类型是不是基本数据类型，我们有以下一些方法：

1. class类提供的 isPrimitive方法

public void testIsPrimitiveType(){  
    System.out.println(int.class.isPrimitive());  
    System.out.println(String.class.isPrimitive());  
    System.out.println(boolean.class.isPrimitive());  
    System.out.println(void.class.isPrimitive());  
	System.out.println(Double.class.isPrimitive()); // false
}

这个方法用于判断Class对象是否为一个基本数据类型和Void，如果是八大基本类型加void，则返回true，否则返回false。

2. class也可以使用isArray()判断是否为数组类型

public void testIsArray(){  
    int[] ints = new int[0];  
    System.out.println(ints.getClass().isArray());  
    System.out.println("1".getClass().isArray());  
}

3. getName

  • 引用类型不是数组类型 返回二进制名称
  • 基本类型或者void，返回的是一个String字符串
  • 数组类型，由多个或一个[后接内部形式的元素类型的名称组成。
@Test  
public void testGetName(){  
    MergeSort mergeSort = new MergeSort();  
    System.out.println(mergeSort.getClass().getName()); // us.laelaps.sort.MergeSort  
    System.out.println(int.class.getName());            // int  
    System.out.println(String.class.getName());         // java.lang.String  
    System.out.println(void.class.getName());           // void  
    int[] ints = new int[1];  
    System.out.println(ints.getClass().getName());      // [I  
    String[][] strings = new String[2][3];  
    System.out.println(strings);                        // [[Ljava.lang.String;@4bb4de6a  
}

4. 判断数组中的类型getComponentType()

使用getComponentType() 方法返回数组中每个元素的类型，如果不表示数组，则返回null

@Test  
public void testGetComponentType(){  
    int[] ints = new int[1];  
    String[][] strings = new String[2][3];  
    double[] doubles = new double[2];  
    System.out.println(ints.getClass().getComponentType());     // int  
    System.out.println(doubles.getClass().getComponentType());  // double  
    System.out.println(strings.getClass().getComponentType());  // class [Ljava.lang.String;  
    System.out.println(int.class.getComponentType());           // null  
}

5. 使用Apache Commons Lang3 中的 ClassUtils.isPrimitiveOrWrapper

他用来判断给定的类是否属于基本类型或者是基本类型的包装类。

@Test  
public void testApacheCommons(){  
    System.out.println(ClassUtils.isPrimitiveOrWrapper(int.class));  
    System.out.println(ClassUtils.isPrimitiveWrapper(int.class));  
    System.out.println(ClassUtils.isPrimitiveOrWrapper(double.class));  
    System.out.println(ClassUtils.isPrimitiveWrapper(double.class));  
    System.out.println(ClassUtils.isPrimitiveOrWrapper(MergeSort.class));  
    System.out.println(ClassUtils.isPrimitiveWrapper(MergeSort.class));  
    System.out.println(ClassUtils.isPrimitiveOrWrapper(Double.class));  
    System.out.println(ClassUtils.isPrimitiveWrapper(Double.class));  
}

6. 使用Hutool的ClassUtil.isBasicType

isBasicType 判断是否为基本类型，包括包装类和原始类

@Test  
public void testHutool(){  
    System.out.println(ClassUtil.isBasicType(int.class));  
    System.out.println(ClassUtil.isBasicType(Integer.class));  
    System.out.println(ClassUtil.isPrimitiveWrapper(Integer.class));  
    System.out.println(ClassUtil.isBasicType(MergeSort.class));  
}

Java 二进制名称（Binary Name）

根据Java语言规范，二进制名称用来表示类，接口，数组和原始类型的规范化字符串的表达形式。

  1. 类和接口的二进制名称：
    • 类和接口的二进制是完全限定名的内部形式
    • 如果由多个标识符组成，使用.分割。最后的标识符表示接口或者类的简单名称，其他的标识符表示为包名
    • 内部类的二进制名称使用$分隔
    us.laelaps.OuterClass
us.laelaps.OuterClass$InnerClass
  2. 数组类型的二进制名称：
    • 数组的二进制名称由一个或多个[和基础类型或引用类型的二进制名称组成；通过使用方括号的数量，我们可以区分不同纬度的数组。
