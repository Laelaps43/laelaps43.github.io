en….没什么好说的，最简单的二分搜索，但是看别人的写法，Scala挺有意思的，递归的写法，但是本质没什么区别。 Java：

public static int search(int[] nums, int target) {  
    int minIndex = 0;  
    int maxIndex = nums.length - 1;  
    while(minIndex <= maxIndex){  
        int middleIndex = (minIndex + maxIndex) / 2;  
        if (nums[middleIndex] == target){  
            return middleIndex;  
        }else if (nums[middleIndex] < target){  
            minIndex = middleIndex + 1;  
        }else if (nums[middleIndex] > target){  
            maxIndex = middleIndex - 1;  
        }  
    }  
    return -1;  
}

Python：

def search(self, nums: List[int], target: int) -> int:  
    min_index = 0  
    max_index = len(nums) - 1  
    while min_index <= max_index:  
        middle_index = (min_index + max_index) // 2  
        middle_value = nums[middle_index]  
        if middle_value == target:  
            return middle_index  
        elif middle_value < target:  
            min_index = middle_index + 1  
        elif middle_value > target:  
            max_index = middle_index - 1  
    return -1

Scala:

import scala.annotation.tailrec

  

def search(nums: Array[Int], target: Int): Int = {
return helper(nums, target, 0, nums.length - 1)
}

  

@tailrec
def helper(nums: Array[Int], target: Int, minIndex: Int, maxIndex: Int): Int = {
	val middleIndex = (minIndex + maxIndex) / 2
	
	if (minIndex > maxIndex) return -1
	
	nums(middleIndex) match
		case x if x == target => middleIndex
		case x if x < target => helper(nums, target, middleIndex + 1, maxIndex)
		case _ => helper(nums, target, minIndex, middleIndex - 1)
}

Scala@tailrec用于标记一个函数为尾递归，尾递归是递归是最后一步操作，递归完之后不能有任何的操作，尾递归不会添加新的栈帧，因此不会出现栈溢出的情况。 match为模式匹配，对应Java中的switch。
